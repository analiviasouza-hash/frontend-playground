# Frontend Playground

Repositório de Ana Livia Souza de Freitas com portfolio pessoal e painel admin para edição de conteúdo.

## Estrutura

```
frontend-playground/
├── index.html       — página inicial
├── style.css        — estilos da página inicial
├── portfolio.html   — portfolio pessoal
├── perfil.html      — página de perfil
└── admin.html       — painel admin para editar o portfolio
```

## Como usar

Abra qualquer arquivo `.html` diretamente no navegador.

### Admin

Acesse `admin.html` para editar as informações do portfolio (nome, cargo, sobre, habilidades, projetos e contato). As alterações são salvas automaticamente no navegador via localStorage e refletidas em `portfolio.html`.
